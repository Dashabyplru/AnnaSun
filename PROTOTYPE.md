# Прототип веб-сайта AnnaSun
## Техническое задание на дизайн-прототип

### 1. Общие принципы
- **Стиль**: Эко-минимализм с элементами рустик-шика
- **Цветовая палитра**:
  - Основной: #7c9a5e (натуральный зелёный)
  - Тёмный: #5d7a42 (изумрудно-зелёный)
  - Светлый: #a8c889 (салатовый)
  - Акцентный: #d4a574 (песочный/карамельный)
  - Фон: #faf8f5 (молочно-белый)
  - Текст: #2c2c2c (почти чёрный)
  - Вторичный текст: #6e6e6e (серый)
- **Типографика**: Sans-serif (Segoe UI, системный шрифт)
- **Размеры**: Адаптивная сетка 1200px контейнер

### 2. Страницы и экраны

#### 2.1 Главная страница (index.html)
- **Header**: Логотип (зелёный круг с каплей/листочком), навигация, корзина
- **Hero**: Градиентный фон (светло-зелёный → песочный), заголовок 3rem, подзаголовок, CTA-кнопка
- **Преимущества**: 4 карточки в сетке, иконки SVG (листок, сердце, щит, звезда)
- **Популярные товары**: 6 карточек товаров в сетке 3×2
- **О бренде**: Двухколоночный блок с текстом и изображением
- **Footer**: 3 колонки, тёмный фон

#### 2.2 Каталог (catalog.html)
- **Sidebar фильтров**: Белая карточка, скруглённая
  - Категории (чекбоксы): Уход за собой, Натуральное, Для мужчин, Скрабы, Премиум, Для ванны
  - Знаки зодиака (12 чекбоксов)
  - Цена: два инпута От/До
  - Кнопки: Применить (зелёная), Сбросить (контурная)
- **Сетка товаров**: 3-4 колонки на десктопе, 2 на планшете, 1 на мобильном

#### 2.3 Карточка товара (product.html)
- **Двухколоночный layout**:
  - Левая: Изображение 100% ширины, скруглённое
  - Правая: Название 2rem, цена 1.5rem, описание, мета-информация, теги, выбор количества, кнопка "В корзину"
- **META-блок**: Состав, вес, знак зодиака, образ жизни

#### 2.4 Корзина (cart.html)
- **Таблица**: Товар | Количество | Сумма | Действие
- **Итого**: Жирный текст, зелёный цвет
- **Кнопки**: Оформить заказ (зелёная), Продолжить покупки (контурная)

#### 2.5 О компании (about.html)
- **Два больших блока**: Текст + изображение
- **Принципы**: 4 карточки с иконками

#### 2.6 Контакты (contacts.html)
- **3 карточки**: Адрес, телефон, email (иконки SVG)
- **Форма**: Имя, телефон, email, сообщение, кнопка отправки

#### 2.7 Оплата (payment.html)
- **4 карточки способов оплаты**: Наличные, Карта, Онлайн, Рассрочка
- **Блок безопасности**: Текст с серым фоном

#### 2.8 Доставка (delivery.html)
- **4 карточки способов доставки**: Курьер, Почта, СДЭК, Самовывоз
- **Блок "Важно знать"**: Список важной информации

### 3. Компоненты дизайна

#### 3.1 Кнопки
- **Primary**: Зелёный фон, белый текст, скругление 12px, hover: тёмно-зелёный, тень
- **Outline**: Прозрачный фон, зелёная граница 2px, hover: заливка зелёным
- **Размеры**: padding 12px 30px, font-size 1rem

#### 3.2 Карточки товаров
- Белый фон, скругление 12px
- Тень: 0 2px 8px rgba(0,0,0,0.06)
- Hover: transform translateY(-5px), тень усиливается
- Изображение: 280px высота, object-fit cover
- Теги: скругление 20px, песочный фон

#### 3.3 Формы
- Border: 1px solid #e0d8ce
- Скругление: 8px
- Padding: 12px
- Focus: border-color #7c9a5e

#### 3.4 Адаптивность
- **Desktop**: > 1024px — полная сетка
- **Tablet**: 768px — 1024px — 2 колонки
- **Mobile**: < 768px — 1 колонка, мобильное меню
- **Small mobile**: < 480px — уменьшенные отступы

### 4. Иконки (SVG)
- logo: Зелёный круг с каплей/листочком
- cart: Корзина покупок
- phone, email, location: Контакты
- search: Поиск
- menu: Гамбургер для мобильного меню
- leaf, heart, shield, star: Преимущества
- truck, payment: Доставка и оплата
- natural: Листочек
- empty-cart: Пустая корзина
- moon, fire: Дополнительные иконки

### 5. Интеракции
- **Hover эффекты**: Кнопки, карточки товаров, ссылки навигации (подчёркивание)
- **Анимации**: slideIn для уведомлений, smooth scroll
- **Mobile menu**: Переключение класса active
- **Фильтры**: Применение без перезагрузки страницы
- **Корзина**: Добавление/удаление, подсчёт суммы, localStorage

### 6. Требования к реализации
- Семантические теги HTML5
- SVG-спрайт для иконок
- SCSS с переменными и вложенностью
- Адаптивная сетка CSS Grid + Flexbox
- JavaScript для динамического контента
- localStorage для корзины
